# ML_mini_projects
# ML Mini Projects Web App 🚀

This project is a **Machine Learning mini-projects web application** built using **Python and Streamlit**.  
It provides an interactive interface where users can explore and interact with different machine learning models.

🌐 **Live Demo:**  
https://ml-mini-projects.onrender.com/

---

## 📌 Features

- Interactive **Streamlit web interface**
- Multiple **Machine Learning mini projects**
- Real-time **model predictions**
- Simple and user-friendly UI
- Deployed online using **Render**

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Pickle

---

## 📂 Project Structure

```
ml-mini-projects/
│
├── app.py              # Main Streamlit application
├── models/             # Saved machine learning models (.pkl)
├── dataset/            # Dataset files
├── requirements.txt    # Project dependencies
└── README.md           # Project documentation
```

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ml-mini-projects.git
```

Go to the project directory:

```bash
cd ml-mini-projects
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

Run the Streamlit application using:

```bash
streamlit run app.py
```

Then open your browser and go to:

```
http://localhost:8501
```
